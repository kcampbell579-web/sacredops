# SacredOps — Facebook / Instagram Ad Kit

Three angles, one free-demo destination. Run all three to the **same ICP audience**
(construction/warehouse; owner, VP Ops, safety/operations titles; US; age 35–60) and
let CPL decide the winner. Objective: **Leads** or **Traffic** (optimize for landing
page views, not link clicks). Placements: **Facebook Feed + Instagram Feed only** —
turn OFF Audience Network and Reels/Stories for the first test.

Each ad image is 1080×1350 (4:5, best feed ratio): `ad-supervisors.png`, `ad-crews.png`,
`ad-incident.png`.

## Launch checklist
1. One campaign → three ad sets (or one ad set, three ads) → same audience.
2. Repoint each ad's destination URL to its landing page above (NOT the pricing/buy page).
3. Placements: Facebook Feed + Instagram Feed only. Kill Audience Network.
4. Budget: start ~$20–40/day total across the three; let it run 3–4 days before judging.
5. Watch in GA4 or Ads Manager: cost per **landing page view** first, then demo signups.
   The Meta Pixel fires `ViewSupervisorProduct` / `ViewWorkerProduct` / `ViewIncidentProduct`
   on each page, and `Lead` when someone starts the demo.
6. Pour budget into the lowest cost-per-demo-signup page; pause the losers.

_Note: the caution-tape "PROPRIETARY" band is a brand/anti-copy visual, not legal IP
protection. It looks sharp in feed and matches the landing pages — that's its job._
